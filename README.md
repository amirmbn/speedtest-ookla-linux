# speedtest-ookla-linux
One-command Bash installer and runner for Ookla Speedtest CLI on Linux.
<br><br>
<div align="right">

- کد زیر رو در سرور اوبونتو وارد و تا اتمام نصب صبر کنید، بعد از اجرا شما باید آیدی مقصد رو وارد کنید.
</div>
<br>
<div align="left">

```
mkdir -p speedtest && cd speedtest && wget https://raw.githubusercontent.com/amirmbn/speedtest-ookla-linux/main/speedtest-ookla.sh && chmod +x speedtest-ookla.sh && sudo ./speedtest-ookla.sh
```
</div>
<div align="right">
- در ادامه لیست مقصد های speed test براتون قرار داده خواهد شد
</div>
